# 구글 첫 화면 클론

> 사용자가 직접 북마크를 추가하고, 구글 검색이 가능한 가벼운 웹페이지

## 주요 기능

- 구글 검색창을 통한 검색 연동
- 사용자가 입력한 URL을 북마크로 등록
- favicon 자동 추출
- localStorage를 이용한 상태 저장
